# ⚠️ Development Tap - TestPyPI Distribution

This is a **development tap** for testing respec-ai before production release.
Packages are pulled from TestPyPI and may be unstable.

**Production users should wait for the official PyPI release.**

## Installation

```bash
# Add the development tap
brew tap mmcclatchy/respec-ai

# Install respec-ai (development version)
brew install respec-ai
```

## Development Notice

This formula installs from TestPyPI and is for testing purposes only.

- Not recommended for production use
- Features may be unstable
- Breaking changes may occur between versions
